## 🧑🏻‍💻 수원대학교 재학생 중고거래 서비스

### 개발 동기
- 교내에서 어떠한 서비스를 진행하게 된다면, 확보된 유저 타겟이 가능하다고 판단.
- 학생들이 재학생 인증을 거친 후에 회원가입이 된다면, 신뢰있는 중고거래 서비스를 제공이 가능하다고 판단.
- 대학교 교내 강의들의 서적, 기숙사 or 자취생들끼리의 생필품 거래 등 학생들끼리 저렴하게 사고 파는 시스템이 필요하다고 판단.
